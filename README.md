# Meme Generator — Image Upload

Simple Next.js + Tailwind starter focusing on image upload (drag-and-drop or click-to-select) and preview.

Quick start

1. Install dependencies

```
npm install
```

2. Run development server

```
npm run dev
```

Open http://localhost:3000
# Neon Snake

A simple single-player Snake game using HTML Canvas and vanilla JavaScript.

- Controls: Arrow keys (or WASD)
- Dark background, neon green snake, red food dots
- Score shown at the top-right
- Game over screen with restart button when you hit the wall

To run locally:

1. Open `index.html` in your browser (double-click or serve with a simple HTTP server):

```bash
# from repository root
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
